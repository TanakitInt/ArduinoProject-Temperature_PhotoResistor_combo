# Arduino - Temperature and Brightness
Messure a temperature and brightness at the moment AT ONCE!

## Components used :
x1 Breadboard  
x1 Arduino Mega (China Model)  
x1 USB-B Cable  

x1 Thermistor  
x1 Photo Resistor  
x2 10K Ohm Resistor  
x1 330 Ohm Resistor  
x1 LED (Any color)
